# Snake-Game-Repo
Repo containing Source code for the Snake Game Development coursework

# Project Overview:
A beautifully designed Snake game with a magical garden theme, built with Pygame. 
Control a friendly snake as it collects enchanted fruits while avoiding mystical barriers in an enchanted garden.

# Core Gameplay
Intuitive Controls: Arrow key navigation with smooth movement

Progressive Difficulty: Speed increases as you collect more fruits

Level System: 5 levels with increasing obstacles

Score Tracking: High score persistence with level multipliers

Win Condition: Master the garden by reaching Level 5

# Visual Design
Magical Garden Theme: Enchanted fruits, mystical barriers, garden aesthetics

Animated Elements: Sparkling food, snake with directional eyes, color gradients

Professional UI: Start menu, game HUD, instructions, game over screens

Smooth Animations: Frame-based movement with visual feedback

# User Interface
Start Menu: Play, Instructions, and Quit options

Game HUD: Real-time score, level, length, and speed indicators

Game Over Screen: Final score with restart option

Instructions: Comprehensive game guide

# Technical Features
Modular Architecture: Clean, object-oriented design

Collision Detection: Precise food, obstacle, and self-collision

Sound Effects: Placeholder sounds for game events

Extensible Code: Easy to add new features and levels

# Quick Start
Prerequisites
Python 3.8 or higher

Pygame 2.5 or higher

# Installation
Clone the repository

Install Pygame:
bash
pip install pygame

Run the game:
bash
python main.py
